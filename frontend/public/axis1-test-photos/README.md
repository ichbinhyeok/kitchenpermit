# Axis 1 Test Photos

Local-only test fixtures for Hood Closeout Proof Link QA.

These images were downloaded from Wikimedia Commons for development testing. They are not product defaults, brand assets, or commercial sample content. See `manifest.json` for source pages, original URLs, image dimensions, and license metadata.

Case grouping:

- `clean_closeout`: completed-service proof and clean/reset fixtures.
- `blocked_access`: dirty or incomplete-path fixtures for no-clean-claim and written-record testing.
- `condition_review`: rooftop fan, grease path, and recorded-condition fixtures.
- `web-fixtures/`: broader local-only web scrape set for rough vendor/customer flow testing. These are intentionally not curated as product sample assets.
- `local-scrape/`: larger gitignored Bing thumbnail scrape for local-only data-volume testing.
